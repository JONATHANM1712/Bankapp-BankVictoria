Tentang Aplikasi

Aplikasi ini adalah latihan magang pertama untuk membuat sistem input data nasabah. Fitur utama meliputi tambah data, edit data, hapus data, dan tampilan daftar nasabah.

Data yang digunakan bersifat fiktif untuk kebutuhan pembelajaran. Aplikasi ini dapat dikembangkan menjadi sistem banking sederhana dengan fitur transaksi, pencarian nasabah, dan laporan data.
